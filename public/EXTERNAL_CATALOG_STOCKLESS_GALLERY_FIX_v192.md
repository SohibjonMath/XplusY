# OrzuMall v192 — Xitoy katalogi qoldiqsiz va mobil galereya tuzatishi

- Xitoy katalogi ombor qoldig‘i bilan cheklanmaydi.
- Buyurtma serveri tashqi katalog mahsulotlarida supplier stock qiymatini hisobga olmaydi.
- Admin import oynasidan qoldiq inputi olib tashlandi.
- Mobil thumbnail lentalari gorizontal scroll oladi.
- Aktiv thumbnail faqat lentaning ichida scroll qilinadi; butun sahifa siljimaydi.
- Xitoy rasmiga berilgan sun’iy zoom olib tashlandi, katta rasm to‘liq `contain` rejimida ko‘rinadi.
