# API’siz 1688 importer v164

1. Sayt ZIP faylini Netlify’ga deploy qiling.
2. Chrome kengaytmasining eski versiyasini olib tashlang.
3. `OrzuMall_1688_Importer_v2_0_VISIBLE_VARIANT_BRIDGE.zip` faylini arxivdan chiqaring.
4. `chrome://extensions` → Developer mode → Load unpacked orqali `chrome-extension-orzumall-1688` papkasini tanlang.
5. 1688 akkauntiga kiring. Login qilinmasa 1688 barcha spetsifikatsiyalarni sahifaga yuklamasligi mumkin.
6. Mahsulot sahifasini yangilang va `OrzuMall’ga import` tugmasini bosing.
7. Admin oynasida rang, o‘lcham/model va SKU sonini tekshirib saqlang.
