# OrzuMall v175 — Foydalanuvchilar markazi

- Admin panelga alohida `Foydalanuvchilar` bo‘limi qo‘shildi.
- Authenticated mijozlar uchun server-side presence heartbeat ishlaydi.
- Online holat oxirgi 125 soniyalik faollik asosida hisoblanadi.
- Admin jami foydalanuvchilar, online soni, faol mijozlar, umumiy balans va tasdiqlangan xarid summasini ko‘radi.
- Profil, telefon, balans, Telegram ID, admin izohi va xavfsiz qo‘shimcha JSON maydonlarini boshqarish mumkin.
- Eski parol hech qachon ko‘rsatilmaydi; yangi vaqtinchalik parol o‘rnatish mumkin.
- Muhim o‘zgarishlar `admin_user_audit` jurnaliga yoziladi.
