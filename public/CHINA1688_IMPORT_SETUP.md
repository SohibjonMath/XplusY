# OrzuMall v161 — 1688 xavfsiz galereya va SKU importi

## Nima tuzatildi
- Galereya butun sahifadan yig‘ilmaydi. Faqat chap media blok va uning miniatyuralari olinadi.
- Sharhlar, foydalanuvchi yuklagan suratlar, hujjatlar, xizmat ikonkalari va rang ikonkalari asosiy galereyadan chiqarildi.
- Strukturali SKU ma’lumotida begona qator bo‘lsa, u tashlab yuboriladi.
- Toza strukturali SKU bo‘lmasa, ko‘rinadigan variantlardan xavfsiz SKU kombinatsiyasi yaratiladi.
- Server ham SKU bo‘sh qolsa admin tasdiqlagan variantlardan zaxira kombinatsiyalarni yaratadi.

## O‘rnatish
1. Sayt ZIP faylini Netlify’ga deploy qiling.
2. Chrome’da `chrome://extensions` sahifasini oching.
3. Eski importer kengaytmasini olib tashlang.
4. `OrzuMall_1688_Importer_v1_8_SAFE_MEDIA_VARIANTS.zip` faylini ochib, ichidagi papkani `Load unpacked` bilan ulang.
5. Versiya `1.8.0` ekanini tekshiring.
6. 1688 sahifasini yangilang va qayta import qiling.

## Tekshirish
Import oynasida galereya rasmlari va variantlar alohida ko‘rinadi. Keraksiz rasm qolsa belgisini olib tashlab saqlang.
