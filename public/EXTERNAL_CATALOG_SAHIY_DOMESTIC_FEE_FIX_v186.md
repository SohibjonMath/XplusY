# OrzuMall v186 — Sahiy ichki yetkazish summasi fix

- Sahiy sahifasidagi “Xitoy ichida yetkazish” summasi endi uch bosqichda aniqlanadi: ko‘rinadigan label kartasi, umumiy sahifa matni va page-state scriptlari.
- Label bilan summa orasidagi DOM qatlamlar, ikki nuqta va dekorativ belgilar regexni buzmaydi.
- Importer qiymatni topolmasa admin oynasi jim turib `0` qabul qilmaydi: maydon sariq ogohlantirish bilan ajraladi.
- Manba narxi, ichki yetkazish summasi va 7 kg dan ortiq yuk haqi landed-cost hisobiga qo‘shiladi.
